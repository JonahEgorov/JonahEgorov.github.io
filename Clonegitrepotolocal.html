Clone GitHub Repo to Local System

I had fun with this assignment, I was not familiar with the Git Shell however I was familiar with the Desktop application so it was helpful in understanding what the codes were actually doing. At first it seemed daunting that there were so many different commands but it wasn't too hard to understand and there are plenty of cheat sheets out there if I forget what a command does. It seems a little weird navigating through folders without seeing an image of where you are but I can see the potential and the shell feels faster than the Desktop Application. I had been using GitHub locally for last week's project and it is much easier just to make all the changes comfortable in the editor of your choice.
