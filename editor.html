Editor

I decided to go with Sublime Text 3 because I used it last semester and it offers pretty much everything I should need for this class. Some things that I thought were cool about Sublime Text was that it is available for Windows, Mac, and Linux. It even features a distraction free mode which I thought was very interesting. Another feature was the ability top change the color of the editor depending if your using it in a dark room or not. I have used Sublime in the past when I was making a website for someone to save html code offline that I could reference later. I was able to create the google Drive folder using it. It was pretty simple to set up, a little confusing at first but not too hard to figure out.
