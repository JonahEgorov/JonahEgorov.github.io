SEO for eCommerce 2

Utilizing best SEO Practices

Do not have duplicate content. It is bad for the same content. It could lead to being counted as SPAM in google and blacklisted. Make sure the tab at top is different. Write clear and unique page titles for each product page on the site. Don’t copy and paste the manufacture of the product description page into your site. You can tell the Search Engine where the content come from. Make sure to tell google which page is original. Keyword Density refers to total number of keywords divided by words on the page. Make sure the keyword words are more natural. Cloaking Text is not very good idea. Avoid the practice at all cost. The key to keywords is using them naturally. There are great ways to use headings and keywords to make your website more SEO friendly. Make sure the load times of media is not too big. Make sure the file size is smaller so it loads faster on the website. There are free tools around the web that can make pictures smaller for the web site. Make sure your website works on all devices. Make sure to set a goal. Measure how well your business is done. SEO measurement is a little more involved. Google analytics is a great tool.

Implementing on-Page Logic

Try to find a domain name that relates to their industry. Like musethemes.com Three is the maximum to have for you url. Avoid top level domains. .cc .info and .bc can trigger spam filters. Make the url so people can hear it and know how to spell it. Use hyphens scarcely. Shorter is better. Use only one hyphen max and only if required. Your home page should explain what your page is about. Focus only on benefits. Encourage deeper exploration. Make it clear what you do or sell. Make sure the important stuff is above the fold. Keep the page light and easy to skim. Match the tone with the design and user. Storefront needs to be friendly and approachable. Having Mobile Accessibility will really help if they are on mobile. Mobile friendly testing tool is important. There is a google tool to test if it is mobile friendly. Responsive design is important. It will show what the mobile website looks like. Design with mobile usability in design. Average size is font size is 17px.

Incorporating Measurement and Optimization Tools

Learning how to test design tweaks on your site is very important. AB Testing is also called Split Testing. Optimizely is a good tool for the job. You cans how two version of your site at once. You can compare which version of your site works the best. YOu can set up experiments using these types of tools. (This is really cool). YOu can test design changes in one variation or multiple variations. You can't set up a specific audience to see what audience sees the experience. Optimizely is a pretty straight forward tool to use for Ab testing/Split Testing. Goals are used to see which version of your site performs better. Google analytics is also very helpful. You can drill down to where the users are coming from and you can even see their city. Heat mapping tools are helpful to see where they are and where they scroll and see what gets the most attention. Moz is a suite of SEO project designed for bigger sites. It makes measuring the impact extremely easy. It’;s expensive but worth the time on big sites or just do the free 30 day trial. Don’t make changes to often. Give them a chance to change and be patient.
